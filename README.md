## 🎯 Project Overview

**SupplyPulse** is an Executive Supply Chain Analytics Dashboard developed in **Power BI** to provide end-to-end visibility into supply chain operations. The solution enables organizations to monitor inventory performance, evaluate supplier efficiency, analyze transportation costs, and track delivery KPIs through an interactive and data-driven reporting experience.

---

## 🚀 Key Features

✅ Executive KPI Monitoring

✅ Inventory Performance Analysis

✅ Supplier Performance Tracking

✅ Transportation Cost Analysis

✅ Sales Quantity Monitoring

✅ Stock Availability Tracking

✅ On-Time Delivery Analytics

✅ Interactive Filters & Drilldowns

✅ Warehouse-Level Insights

✅ Category-Wise Performance Analysis

---

## 📊 Key Business Metrics

| KPI | Description |
|------|------------|
| 🚚 Transportation Cost | Total logistics expenditure |
| 📦 Total Inventory | Inventory across warehouses |
| 📈 Sales Quantity | Total units sold |
| ⏱️ On-Time Delivery % | Delivery performance indicator |
| ✅ On-Time Deliveries | Successful deliveries within schedule |
| 📋 Stock Availability % | Inventory readiness metric |
| 🔴 Out-of-Stock Products | Products requiring replenishment |
| ⌛ Average Delivery Time | Supplier responsiveness |
